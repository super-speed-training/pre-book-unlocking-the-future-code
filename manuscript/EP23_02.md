# EP 23-02 Azure web app + app insight

### สร้าง web app 

สร้าง web app เพื่อใช้ในการ pubish web ที่อยู่ใน Local ของเราให้ไปเป็น online 

ในเว็บ [Microsoft Azure](https://portal.azure.com/) ให้คลิ๊กเข้าไปที่ App Services
